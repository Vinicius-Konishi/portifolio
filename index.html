Python

Raspagem de Dados (Web Scraping): Python é amplamente usado para coletar dados de websites de forma automatizada. Ferramentas como BeautifulSoup e Scrapy permitem a extração de informações estruturadas de páginas HTML, ajudando na coleta de grandes volumes de dados.

HTML

Estruturação de Conteúdo Web: HTML é utilizado para criar a estrutura básica de qualquer página web. Ele define a organização do conteúdo, como cabeçalhos, parágrafos, listas, links, imagens e formulários, sendo essencial para qualquer projeto web.

CSS

Design Responsivo: CSS, com o uso de media queries, permite criar layouts que se adaptam a diferentes tamanhos de tela, como smartphones, tablets e desktops. Frameworks como Bootstrap ajudam a implementar design responsivo rapidamente.

MySQL

Gerenciamento de Banco de Dados Relacional: MySQL é amplamente utilizado para armazenar e organizar dados em tabelas relacionais. Ele permite criar relacionamentos entre tabelas usando chaves primárias e estrangeiras, garantindo a integridade dos dados.

Demoquerycy

O projeto DemoQuerycy visa melhorar o acesso a informação digital sobre os vereadores da cidade de São José dos Campos. Tendo em vista que o site da Camara sem papel possui muitos filtros e informações desnecessárias para o usuário, o DemoQuerycy é uma forma simplificada de visualizar os dados parlamentares de maneira rápida e prática. Composto por 8 alunos do 1° Semestre de DSM o projeto segue em desenvolvimento utilizando as tecnologias Python, HTML 5, CSS 3, MySQL, Flask e GitHub como ferramenta de versionamento.
